Arrays e Algoritmos

Existem alguns pilares quando falamos sobre Algoritmos:

O que é um Algoritmo? É uma programação onde realiza-se passo a passo com o objetivo terminar ou tarefa, automatizar, etc.

Um algoritmo possui 4 caracterpisticas:

Abstração: Entendimento do mundo real em computadiocal.

Modularidade: Fazer cógido reutilizável (Funções Classes)

Estrutura de Dados: int | float | String | Boolean | Listas | Tuplas | Dicionários Z Tuplas

Simpliidade: Manutenível, facilidade em entender outra pessoa a fazer mudanças no código
